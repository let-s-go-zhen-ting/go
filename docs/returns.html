退換貨方式與流程

生效日：2025/10/16　最後更新：2025/10/16

一、可申請情形

  • 收到商品有重大瑕疵、破損或寄錯品。
  • 配件/數量缺漏（以訂單明細為準）。
  • 不適用情形：已拆封之耗材或客製化商品、超過申請期限、非可歸責於本店之損壞等。

二、申請期限

請於到貨 7 日內主動聯繫客服並提供訂單編號、問題描述與清晰照片/影片。

三、處理方式

  • 經確認屬實：可協助換貨、補件，或視情況退款。
  • 退款需依原支付方式或雙方同意之方式辦理。

四、流程

  1. 聯繫客服 → 提供訂單編號與問題證明。
  2. 本店確認 → 告知後續寄回/補寄安排。
  3. 寄回或補寄 → 依指示完成物流處理。
  4. 完成退款或換貨 → 通知結果。

五、運費負擔

  • 因商品瑕疵/寄錯：往返運費由本店負擔。
  • 非瑕疵因素之退貨（若同意受理）：運費需由消費者負擔。

六、備註

海外/預購商品時程與風險較高，若遇延誤或不可抗力，將以公告或客服聯繫為準，敬請見諒。
